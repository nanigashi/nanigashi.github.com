Posted on 2013-02-28

Ruby 2.0 が正式に発表なってたのでrvmで利用できるようにしてみます。

Ruby 2.0.0-p0 リリース

環境

  • OS: Ubuntu 12.04 64bit
  • Ruby: ruby 2.0.0p0 (2013-02-24 revision 39474) [x86_64-linux]
  • rvm: rvm 1.14.3 (stable)
